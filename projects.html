[CASE_STUDY]: IT1515 Table Tennis CCA Website

A responsive recruitment platform for NYP Table Tennis Club

01_PROJECT_SCOPE

Who was the client?

NYP Table Tennis CCA - a student club at Nanyang Polytechnic looking to drive recruitment for Year 1 students.

What were the main goals?

  • Create an engaging online presence to attract new members
  • Display training schedules in an accessible format
  • Streamline the signup process with online registration
  • Showcase club events and achievements

What were the challenges?

  • Designing for a diverse audience (prospective students, parents, current members)
  • Ensuring mobile responsiveness for on-the-go access
  • Creating an intuitive user journey from discovery to signup

Homepage - Welcome section with call-to-action

Training Schedule - Weekly session times & progress tracker

02_DEVELOPMENT_ROLE

What was my role?

Lead Front-End Developer - Responsible for the entire user interface and experience design of the 4-page website.

What was my thinking process?

I approached this project with a user-centric mindset, focusing on:

  • Visual Appeal: Using the club's navy blue and gold color scheme for brand consistency
  • Clear CTAs: Prominent "Join Us" buttons throughout the site
  • Information Hierarchy: Organizing content from general (About) to specific (Training Details)
  • Trust Building: Including testimonials and event photos to showcase club culture

03_WORK_PROCESS

How did I manage and execute the project?

1
Research & Planning

Analyzed similar CCA websites and mapped user journeys

2
Wireframing

Created low-fidelity layouts for all 4 pages

3
Development

Built with HTML5, CSS3, Bootstrap 5, and JavaScript

4
Testing & Refinement

Tested across devices and browsers, gathered feedback

Problems encountered and solved:

  • Challenge: Form validation across different browsers
  • Solution: Implemented custom JavaScript validation with clear error messages
  • Challenge: Table responsiveness on mobile devices
  • Solution: Used Bootstrap's responsive table classes with horizontal scroll

Signup Form - Student registration with validation

Events Page - Upcoming competitions & club activities

04_PROJECT_OUTCOME

Was the project successful?

Yes! The website was successfully deployed and received positive feedback from both the CCA committee and new recruits.

What were the measurable results?

4 Responsive Pages
100% Mobile Friendly
A+ Project Grade

Key achievements:

  • Delivered a professional-grade recruitment site on schedule
  • Demonstrated technical proficiency in front-end development
  • Applied secure coding practices throughout development
  • Created a scalable template for future CCA websites
Technologies Used:
HTML5 CSS3 Bootstrap 5 JavaScript
